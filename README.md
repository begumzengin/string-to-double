# string-to-double
 Homework for BIL458

 ### String to Double Conversion Program

 This is a Python program that converts a string input to a double.

 #### How to Run

 1. Open a terminal or command prompt in the directory where the string_to_double.py file is located.
 2. Run the program using the command python string_to_double.py.
 3. Follow the prompt to enter a string representation of a number.
 4. If the input is a valid number, the program will output the converted double. Otherwise, an error message will be displayed.
